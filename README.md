# Meta Ads AI

Meta Ads AI Analyst er et planlagt produkt som skal hjelpe SMB-er å analysere Meta Ads-data med AI for raskere innsikt og anbefalinger.

Prosjektet vil få både frontend og backend (stack bestemmes i neste fase), med fokus på et enkelt dashboard og AI-drevne anbefalinger.
